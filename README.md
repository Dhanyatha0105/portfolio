# Dhanyatha Corry — Portfolio

A skill-map portfolio website showcasing 35+ projects across 7 domains, built with Next.js (static export), Tailwind CSS, and Framer Motion.

## Domains

- **AI Research** — AGI, RAG, Medical AI
- **AI Security** — Network Security, Blockchain, Cryptography
- **Computer Vision** — 3D Reconstruction, Video Analysis, Edge AI
- **Data Science** — Predictive Analytics, Market Analysis
- **Full Stack Web** — Platforms, Dashboards, Cloud & IoT
- **AI Agents** — Voice Assistants, Enterprise Orchestration
- **Quantum Computing** — Quantum Research, QML, Simulation

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs a static site to `out/`.

## Tech Stack

- Next.js 16 (Static Export)
- Tailwind CSS 4
- Framer Motion
- TypeScript
